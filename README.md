# Canteen Food Ordering System for Unilever

A business analysis and system design project for a canteen ordering solution in a large office environment. This repo contains requirements, process flows, ERD, and UI artifacts.

## Contents
- `Project Solution.docx`: full project proposal and analysis
- `Main UI.docx`: UI wireframes
- `ActivityDiagram1.jpg`: activity diagram
- `ContextDiagram.jpg`: context diagram
- `ERDDiagram1.jpg`: ERD

## Scope
- Requirements and stakeholder analysis
- Process and data modeling
- UI flow for ordering and fulfillment

## How to Use
Open the documents directly in Word or a compatible viewer. Diagrams are provided as images for easy viewing.
